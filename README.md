# ⚖️ LegalEase

LegalEase is a web application built with **Next.js** and **Tailwind CSS** that helps users **summarize legal documents, generate legal content, and convert files between different formats**.

## 🚀 Features
- 📄 Summarize legal documents
- ✍️ Generate legal text
- 🔄 Convert files between formats
- ⚡ Fast and responsive interface

## 🛠 Tech Stack
- Next.js
- Tailwind CSS

## 📦 Installation

```bash
git clone https://github.com/kivoxff/LegalEase.git
cd LegalEase
npm install
npm run dev
